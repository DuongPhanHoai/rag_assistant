# Course Descriptions

## C101 Database Fundamentals
Introduces relational modeling, SQL querying, normalization, and database-backed application patterns. Students complete a SQL project and a midterm exam.

## C102 Python for Analytics
Covers Python data structures, data cleaning, notebook workflows, and simple analytics automation. Students complete programming labs and a final analytics project.

## C103 Statistics for Business
Focuses on descriptive statistics, probability, hypothesis testing, and communicating analytical findings to business stakeholders.

## C104 Software Project Studio
Students work in teams to deliver a small software product using agile practices, version control, demos, and peer review.

## C105 Responsible AI Foundations
Introduces fairness, transparency, AI governance, and responsible deployment practices for analytics and software teams.
